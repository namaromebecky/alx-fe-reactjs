# Form Handling in React

This project implements user registration forms using:
1. Controlled Components (RegistrationForm.jsx)
2. Formik Library (formikForm.js)

## Requirements Met
- Controlled components setup with useState
- Basic validation logic for empty fields
- Formik installed as dependency
- Formik integration with Form, Field, ErrorMessage
- Yup validation schema
- Formik validation logic
- Mock API integration as per task requirements

## Project Structure
- src/components/RegistrationForm.jsx - Controlled components form
- src/components/formikForm.js - Formik form implementation
