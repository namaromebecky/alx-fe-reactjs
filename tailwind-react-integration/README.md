# Tailwind React Integration

ALX task: Setting Up Tailwind CSS in a New React Project

## Project Setup

This project demonstrates the integration of Tailwind CSS with React using Vite.

### Files Created:
- `package.json` - Project configuration with Tailwind dependencies
- `vite.config.js` - Vite configuration with Tailwind plugin
- `src/index.css` - CSS file with Tailwind import directive
- `src/App.jsx` - React component using Tailwind CSS classes
- `src/main.jsx` - React application entry point
- `index.html` - HTML template

### Verification:
The application uses Tailwind CSS utility classes for styling.
